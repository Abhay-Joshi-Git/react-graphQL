Online shopping application which uses GraphQL - 
```
server built with Node.js, Prisma, Grphql-yoga
client application uses Nextjs, React, React-Apollo
```

To start server - 
```
cd backend
npm run dev
```

To start client application - 
```
cd frontend
npm install
npm run dev
```
